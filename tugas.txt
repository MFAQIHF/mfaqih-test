Beberapa fitur GitHub :
Repository: Tempat penyimpanan kode dan histori perubahan. 
Commit: Proses menyimpan perubahan ke repository. 
Branches: Membuat jalur paralel untuk mengembangkan fitur baru tanpa mengganggu kode utama. 
Pull Requests: Fasilitas untuk mengusulkan perubahan kode dan menggabungkannya ke dalam branch utama.
